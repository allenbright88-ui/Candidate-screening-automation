# Candidate Evaluation & Screening Workflow

##  What It Does

An automated recruitment pipeline that 
systematically screens and evaluates incoming 
candidates against defined criteria — reducing 
manual HR effort and improving hiring pipeline 
speed significantly.

---

##  How It Works

1. Candidate application received via form or email
2. Data extracted and formatted automatically
3. Application evaluated against defined criteria
4. Candidate scored and categorised
5. Qualified candidates routed to appropriate team member
6. Rejection notifications sent automatically
7. All records logged to Google Sheets

---

##  Tools Used

- **n8n** — Workflow automation
- **Google Sheets** — Candidate database
- **Gmail** — Notifications
